# FoodWise-AI
Öğrenci hayatı kaostur; FoodWise AI bu kaosu Gemini ile yönetir. 🎓 Yoğun akademik tempoya göre kişiselleşen haftalık beslenme planı, bütçe dostu akıllı market listesi ve gıdalar bozulmadan devreye giren proaktif uyarı sinyalleri.  Eldeki malzemelerle saniyeler içinde "kurtarıcı" tarifler üreten, israfı sıfırlayan dijital mutfak asistanın!
